1. Introduction
   
Prompt Engineering is the practice of designing and refining input instructions — called prompts — to elicit
accurate, relevant, and high-quality responses from Large Language Models (LLMs) such as GPT, Claude, and
Gemini. A prompt is the gateway between human intent and machine output. The quality, structure, and framing of
a prompt can dramatically alter the usefulness of the AI response.
This report examines four foundational prompt engineering techniques: Straightforward Prompts, Tabular
Format Prompting, Missing Word Prompting, and Preceding Question Prompting. Each technique is explained with
its definition, core principles, practical examples, use cases, advantages, limitations, and an illustrative diagram.

2. Straightforward Prompts
2.1 Definition & Overview
Straightforward Prompting is the most fundamental technique. It involves providing a clear, direct,
single-sentence or single-paragraph instruction without special formatting or contextual scaffolding. The core
principle is simplicity: ask clearly, receive clearly.
2.2 Core Principles
• Clarity: The prompt must be unambiguous and grammatically correct
• Specificity: Include enough context to constrain the response appropriately
• Conciseness: Avoid unnecessary padding or repetition in the prompt
• Action Orientation: Begin with an action verb: Explain, Translate, Summarize, List, Define

3. Tabular Format Prompting
3.1 Definition & Overview
Tabular Format Prompting is a technique where the user explicitly instructs the AI to present its response as a
structured table with clearly defined rows and columns. This approach is ideal for comparisons, reference data,
and multi-attribute descriptions, leveraging the brain's ability to process tabular data significantly faster than dense
paragraphs.
3.2 Core Principles
• Explicit Format Instruction: Always specify 'in a table' or 'as a table with columns'
• Column Specification: Define the exact columns you need for maximum relevance
• Row Scope: Specify how many items or what category of items to include
• Sortability Intent: Optionally specify ordering (e.g., sorted by year, alphabetically)

4. Missing Word Prompting
4.1 Definition & Overview
Missing Word Prompting (also called Fill-in-the-Blank or Cloze Prompting) provides an incomplete sentence or
template with blanks, asking the AI to fill in missing content based on context and world knowledge. This technique
ensures the response integrates naturally into existing text by constraining AI output to a specific syntactic and
semantic structure.
4.2 Core Principles
• Context Anchoring: Surrounding text provides strong constraints on the answer
• Grammatical Fit: The filled word must match grammatical role (noun, verb, adjective)
• Semantic Coherence: The answer must make logical sense within the full sentence
• Knowledge Retrieval: The model draws from its training data to supply accurate facts

5. Preceding Question Prompting
5.1 Definition & Overview
Preceding Question Prompting (also known as Sequential Questioning or Chain Prompting) asks a series of
related questions in a deliberate sequence, where each subsequent question builds on the context established by
the previous question and answer. This mirrors the Socratic method of inquiry, progressively uncovering deeper
knowledge through targeted questions.
5.2 Core Principles
• Sequential Logic: Questions follow a logical progression from broad to specific
• Context Retention: Each response informs and constrains the next question
• Progressive Depth: Earlier questions establish foundations; later ones build expertise
• Coherent Narrative: The entire sequence maintains a unified topic thread
